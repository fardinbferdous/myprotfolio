# Personal Portfolio

A simple, responsive portfolio website built with plain `HTML`, `CSS`, and `JavaScript`.

## Files

- `index.html` - main page content
- `style.css` - design and layout
- `script.js` - small interactive behavior

## How to customize

Replace these placeholders in `index.html`:

- `Your Name`
- `Your City`
- `your.email@example.com`
- `yourusername` for GitHub and LinkedIn
- project descriptions and skills

## Run locally

Open `index.html` directly in your browser, or start a local server:

```powershell
python -m http.server 5500
```

Then visit: `http://localhost:5500`
